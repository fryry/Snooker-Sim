List of controls

Mouse click Take a shot with desired power 
Mouse movement (leftright) Rotate around the white ball
Up arrow Zoom out from white ball
Down arrow Zoom in to white ball
Left arrow Decrease shot power
Right arrow Increase shot power
I key Throw white ball into air (for demoing)
R key Reset the simulation
Q key Increase mouse rotation speed
W key Reduce mouse rotation speed
1 key Newton’s cradle demonstration
2 key Drop white ball into reds (for demoing)
3 key Spheres bouncing along table (for demoing)
Escape key Close the program
P key Place white ball on cushion 
A key Turn audio on or off
